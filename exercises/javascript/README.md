# Native Array Methods 

High order functions are simply functions that take as an argument another function.

Let's practice using the native array methods `map`, `filter`, and `reduce`!

```
[🌽, 🐮, 🐔].map(cook)
=> [🍿, 🍔, 🍳]

[🍿, 🍔, 🍳].filter(isVegetarian)
=> [🍿, 🍳]

[🍿, 🍳].reduce(eat)
=> 💩
```
